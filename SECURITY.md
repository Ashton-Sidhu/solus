# Security Policy

## Reporting a Vulnerability

Please do not open a public issue for suspected vulnerabilities.

Report security issues privately by emailing the maintainer listed in the repository profile, or by opening a private security advisory if the repository host supports it. Include:

- Affected version or commit
- Steps to reproduce
- Expected impact
- Any relevant logs, screenshots, or proof-of-concept details

We will acknowledge reports as soon as practical and coordinate disclosure once a fix is available.

## Supported Versions

Solus is pre-1.0 software. Security fixes are made on the default branch first.

## Scope

Solus is a local-first desktop app that can run coding agents, shell commands, git operations, local web servers, and filesystem actions on behalf of the user. Treat workspaces, agent sessions, generated artifacts, logs, and local configuration as sensitive unless explicitly shared.
